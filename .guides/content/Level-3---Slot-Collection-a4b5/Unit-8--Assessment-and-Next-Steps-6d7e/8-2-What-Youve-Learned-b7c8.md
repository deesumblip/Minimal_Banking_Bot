## Key Concepts

1. **Slots**, memory for values the user provides.
2. **Slot collection**, `collect:` in flows to ask and store.
3. **Reading slots**, `tracker.get_slot()` in actions.
4. **Ask responses**, `utter_ask_*` when a slot is missing.

## Skills

You can define slots in the domain, collect them in flows, read them in actions, and handle missing or placeholder values in multi-turn dialogue.
