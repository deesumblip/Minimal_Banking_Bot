
Before you build your first Rasa agent, make sure you have what you need to get started.

**Prerequisites**

- **Timing.** This self-guided chapter usually takes about two to three hours.
- **Skills.** You should be comfortable with basic Python, the command line, and editing files in an editor or IDE.
- **License.** Request a free Rasa developer edition license [at this link](https://rasa.com/rasa-pro-developer-edition-license-key-request) and plan to configure **`RASA_LICENSE`** in Lab 0.1.
