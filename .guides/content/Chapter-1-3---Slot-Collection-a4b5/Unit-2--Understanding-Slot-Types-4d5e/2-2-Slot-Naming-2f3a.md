# 2.2 Slot Naming

**Good slot names**:
- `account` — Clear and descriptive
- `user_name` — Descriptive
- `transfer_amount` — Clear purpose

**Bad slot names**:
- `a` — Too vague
- `slot1` — Not descriptive
- `data` — Too generic

**Convention**: Use lowercase, descriptive names. Use underscores for multi-word names.
