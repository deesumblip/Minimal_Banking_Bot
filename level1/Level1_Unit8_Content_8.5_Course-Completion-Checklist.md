### 8.5 Course Completion Checklist

**Why use this checklist**: These items are the minimum you need to have solid before Level 2. If you can do each one, you're not missing a step. You're ready to add actions and keep building on the same agent.

Before moving to Level 2, ensure you can:

- Explain what a response is and where it's defined
- Create a new response in the domain file
- Create a flow that uses responses
- Explain what flow descriptions do
- Train the agent and run Inspector
- Test greet, help, and contact flows
- Describe what Level 1 cannot do and what Level 2 adds
