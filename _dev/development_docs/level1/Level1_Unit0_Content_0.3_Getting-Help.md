### 0.4 Getting Help

**Why we list these**: Setup and YAML can trip you up. Knowing where to look—docs, error messages, and the troubleshooting section—gets you unstuck quickly so you can stay focused on building the agent instead of fighting the environment.

**Stuck?** Here are resources:

- **Rasa Documentation**: [https://rasa.com/docs](https://rasa.com/docs)
- **Error Messages**: Check the troubleshooting section at the end of this guide
- **Common Issues**: Most setup issues are covered in the troubleshooting guide
