**Objective.** In Unit 5 you saw how to train the Level 6 agent. In this lab you will train from the level6 folder and run the assessment.

## Step-by-Step Instructions

**Step 1.** From project root, activate the venv (e.g. `source .venv/bin/activate` on Codio/Linux/macOS, or `.venv\Scripts\activate` on Windows).

**Step 2.** Change to level6: `cd level6`.

**Step 3.** Run training: `rasa train`. Wait for the model to be written to `level6/models/`.

**Step 4.** Run the assessment. The grader will verify that training completed (e.g. model directory or success indicator).

## Running locally

Same steps: activate venv from project root, `cd level6`, `rasa train`, then run the assessment.
